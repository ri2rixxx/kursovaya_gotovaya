# Звуки для Memory Game

Для полного погружения добавьте звуковые эффекты.

## Необходимые звуки:
- flip.wav      - переворот карты
- match.wav     - совпадение карт
- mismatch.wav  - несовпадение карт
- click.wav     - клик по кнопке
- win.wav       - победа в игре
- lose.wav      - поражение

## Форматы:
- Рекомендуемый: .wav
- Также поддерживается: .ogg

## Где взять:
1. Создать самостоятельно (Audacity и др.)
2. Скачать с freesound.org
3. Использовать генераторы звуков

Игра будет работать и без звуков.
