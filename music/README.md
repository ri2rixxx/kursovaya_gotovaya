# Музыка для Memory Game

Фоновая музыка улучшает игровой опыт.

## Треки:
- menu.ogg            - музыка главного меню
- gameplay_easy.ogg   - музыка для легкого уровня
- gameplay_medium.ogg - музыка для среднего уровня
- gameplay_hard.ogg   - музыка для сложного уровня
- game_over.ogg       - музыка окончания игры

## Форматы:
- Рекомендуемый: .ogg (лучшее качество/размер)
- Также: .wav

## Требования:
- Безопасный для всех возрастов
- Негромкая, фоновая
- Зацикленная (loop)

Музыка опциональна.
